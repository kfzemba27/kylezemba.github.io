# GPT Image 2 Prompt — AI Ecosystem Map

> Paste the prompt below directly into ChatGPT with GPT Image 2 selected. You can also try it in the standard ChatGPT interface — it will auto-route to the image model.

---

## THE PROMPT

Create a clean, professional infographic titled "The AI Stack — Tools for Advertising & Marketing." The design should feel like something a top creative agency would produce: dark background (#0D1117 near-black), modern sans-serif typography, flat design with no gradients or decorative effects. Output at 1920x1080 landscape for presentation use.

The infographic is a structured grid of 11 category blocks arranged in 3 columns. Each category block has a colored header label and contains a list of tool names as small rounded pill-shaped chips. All text must be legible.

Use the following exact categories, tool lists, and colors:

**Row 1:**

Block 1 — "LLMs / Thinking Tools" (header color: #818CF8 indigo)
Tools: Claude, ChatGPT, Gemini, Perplexity, Grok

Block 2 — "Image & Video Generation" (header color: #FBBF24 amber) — this block spans 2 columns wide
Tools: Midjourney, Flux, DALL-E, Runway, Sora, Kling, Seedance, Luma, Pika, Nano Banana, Veo 3, ComfyUI

**Row 2:**

Block 3 — "All-in-One Generative AI" (header color: #2DD4BF teal)
Tools: Higgsfield, Freepik, Artlist

Block 4 — "3D & Spatial Generation" (header color: #60A5FA blue)
Tools: Fal / PATINA, Tripo 3D, World Labs, Google Genie

Block 5 — "Audio & Voice" (header color: #FB7185 rose)
Tools: ElevenLabs, Suno, Udio

**Row 3:**

Block 6 — "Copy & Content" (header color: #34D399 emerald)
Tools: Jasper, Copy.ai

Block 7 — "Research & Insights" (header color: #A78BFA violet)
Tools: Perplexity, NotebookLM, ChatGPT Deep Research, Gemini Deep Research

Block 8 — "Workflow & Productivity" (header color: #94A3B8 slate)
Tools: GitHub Copilot, Cursor, Claude Code, Claude Cowork

**Row 4:**

Block 9 — "Presentation & Design" (header color: #FB923C orange)
Tools: Canva AI, Gamma, Figma

Block 10 — "Marketing & CRM AI" (header color: #F87171 red)
Tools: Salesforce Einstein, HubSpot Breeze, HeyGen

Block 11 — "Watch List / Emerging" (header color: #6B7280 gray, dashed border instead of solid) — spans full width
Tools: Mythos, LTX Studio, AI Answer Engine Optimization

**Design specifications:**

- Background: #0D1117 (near-black)
- Category blocks: slightly lighter background (#161B22) with 1px border matching their header color at reduced opacity
- Category header labels: small, all-caps, bold, colored text matching their assigned color
- Tool name pills: rounded rectangles, dark background (#21262D), light gray text (#E6EDF3), small font size, tight padding
- Watch List block: dashed border, muted styling to signal "emerging / not yet mainstream"
- Title at the top: "The AI Stack" in large white bold text, subtitle "Tools for Advertising & Marketing" in smaller muted gray
- Small footnote at the bottom: "★ = Direct hands-on experience" in muted gray, small font
- Tools marked with ★ (add the star symbol after the name): Claude ★, ChatGPT ★, Gemini ★, Perplexity ★, Midjourney ★, Runway ★, Kling ★, Seedance ★, Nano Banana ★, Higgsfield ★

The overall aesthetic should feel like a dark-mode developer dashboard crossed with a premium agency deliverable. Clean grid, high contrast, no clutter. Every tool name must be fully legible. No decorative illustrations, no gradients, no icons — just the text labels, category structure, and color system.

## VARIATIONS TO TRY

If the first output needs refinement, try adding one of these to the end of the prompt:

**For a lighter/white version:**
"Produce a second version on a white background (#FFFFFF) with light gray category blocks (#F8FAFC) and dark text. Same structure, same colors for headers."

**For a tighter layout:**
"Make the grid more compact — smaller pill text, tighter spacing between blocks. Prioritize fitting everything cleanly over generous whitespace."

**For a presentation slide format:**
"Format this as a 16:9 presentation slide. The title should be prominent at the top left, the grid fills the body of the slide, and there is a small logo placeholder bottom right."

---

## NOTE ON LOGOS

GPT Image 2 may attempt to render approximate tool logos inside the pills. If it does and they look inaccurate or cluttered, add this line to the prompt:

"Do not include any logos or icons. Tool names as text only inside the pills."

For the final webpage build, logos will be sourced directly from Clearbit and added programmatically — the image version is for presentation use where text-only pills are cleaner anyway.
