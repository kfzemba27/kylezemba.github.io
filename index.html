A note before the tools
Better context creates better work.

AI is not a replacement for strategy. It is a multiplier for clear direction.

For advertising and marketing work, the advantage comes from the thinking before the output: the audience, the job to be done, the channel, the brand boundaries, and the creative standard you are trying to reach.

01
Context first
Give the tool the same setup you would give a strategist or creative partner: brand, audience, references, tone, mandatories, and what is off-limits.
02
Clarify before making
Ask the model to restate the assignment and identify gaps before it produces work. Misalignment is cheaper to fix before the first draft.
03
Connect the tools
Strong workflows move across text, image, video, audio, and design. One tool rarely owns the full job; the strategy decides the handoff.
04
Iterate with memory
Treat AI work as an evolving collaboration. Reuse brand context, refine based on feedback, and build from what the tool has already learned.

From context to commercial

Example AI Workflow

A step-by-step look at how thinking models, visual references, and Higgsfield Canvas were combined to turn a Nike soccer cleat concept into a generated commercial.

01 / 07
Step 01Context layer

Train Model

ChatGPT 5.5 Thinking • Context Engineering

We started by using ChatGPT 5.5 Thinking as the planning layer. The goal was not to generate randomly, but to train the conversation with the concept, product, creative intent, medium, constraints, and desired output. This created a shared context for the rest of the workflow.

  • Concept definition
  • Commercial medium
  • Product and brand constraints
  • Creative direction
Step 02Tool choice

Research

ChatGPT Research + Tool Selection

Within the same planning chat, we explored which AI tools would best support the project. The workflow eventually centered around Higgsfield Canvas because of its node-based structure, visual references, and ability to combine prompts, images, and video generation in one system.

  • Tool landscape
  • Workflow examples
  • Visual references
  • Higgsfield Canvas selected
Step 03Production map

Structure

Workflow Architecture

We then designed the production pipeline before generating any final media. The structure became: Brand/Product Brief → Mood Board → Commercial Concept → Storyboard → Shot Table → Keyframes → Final Prompt → Video Generation.

  • Brand/Product Brief
  • Mood Board
  • Concept
  • Storyboard
  • Shot Table
  • Keyframes
  • Video Prompt
  • Generation
Step 04Node content

Build

ChatGPT + Claude + Gemini

After the workflow was structured, thinking models were used to build the content for each generating node. Each node had a clear job: define the brand, analyze mood, create the concept, extract the storyboard, format the shot table, compile the final prompt, and prepare the video generation.

  • Prompt systems
  • Node instructions
  • Structured outputs
  • Model collaboration
Step 05Visual system

Visualize

Reference Images + Visual Anchors

To improve consistency, we created visual anchors for the workflow: a mood board, cleat reference, hero player reference, goalie reference, and storyboard/keyframe sheet. These visuals helped lock the theme, characters, product, and cinematic style before video generation.

  • Mood board
  • Cleat reference
  • Hero athlete reference
  • Goalie reference
  • Storyboard sheet
Step 06Canvas execution

Execute

Higgsfield Canvas

Once the planning, prompts, references, and keyframes were in place, the full workflow was executed inside Higgsfield Canvas. The final video node used the compiled prompt, storyboard sheet, character references, and product reference as generation context.

  • Canvas execution
  • Node-based workflow
  • Prompt + references
  • Video generation
Step 07Output

Results

Generated Commercial Output

The final result was a cinematic soccer commercial concept built from a structured AI workflow rather than a single prompt. The process moved from context engineering to visual consistency, then into storyboard-driven generation.
